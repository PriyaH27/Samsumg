# Dubai Smart Real Estate Intelligence Platform Using Machine Learning

An advanced AI-powered platform for Dubai real estate intelligence, featuring machine learning predictions, smart recommendations, and comprehensive property management.

## 🚀 Features Implemented

### Core Modules
1. **User Interface & Property Finder**
   - Budget-based property search
   - Advanced filtering (location, type, size, bedrooms, furnishing)
   - Sorting options (price, size, bedrooms)
   - Property display with images

2. **🤖 Machine Learning Features**
   - **Property Price Prediction**: Random Forest model predicting prices based on location, size, amenities
   - **Smart Property Recommendations**: Personalized suggestions based on budget and preferences
   - **ROI Calculator**: Investment analysis with rental yield and appreciation calculations

3. **🗺️ Smart Map Features**
   - Interactive map showing all properties
   - Location-based property clustering
   - Dubai area intelligence (Downtown, Marina, Jumeirah, etc.)

4. **🏢 Property Management**
   - Add new properties
   - View property database
   - Session-based property storage

5. **🧠 AI Chatbot**
   - Simple NLP responses for common queries
   - Real estate specific answers

6. **📊 Analytics Dashboard**
   - Key metrics (total properties, average price)
   - Price distribution by location
   - Property type analysis

## 🛠️ Tech Stack
- **Frontend**: Streamlit
- **ML**: Scikit-learn, Random Forest
- **Data**: Pandas, NumPy
- **Maps**: Folium
- **Language**: Python
